# Formation des équipes  

Vous allez travailler en équipe de 3 à 4 personnes durant toute la session.  

Les équipes resteront les mêmes pour tous les problèmes.  

## Comment former les équipes?

Les équipes ont été formées de manière aléatoire par l'enseignant.  Consultez la page de [billetrie](billeterie.md) pour voir votre équipe.  

!!! note
    Vous devez utiliser votre compte **Microsoft du Cégep** pour vous connecter.
